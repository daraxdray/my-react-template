File Structure

After downloading the template you will find 2 files in it, project source and documentation. We have adjusted the directory structure from components, base components, layouts, libraries, and espicially frontend assets for production use, so you can quickly use this starter kit for your project.

The file tree below will describes more clearly about the file structure.
